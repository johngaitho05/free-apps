Description


Having trouble sending mails in Odoo? Are you getting an error like... "SMTPServerDisconnected: please run connect() first"? Then worry no more as this module will solve your issue. Usually some mail clients like zoho requires the sender to match with the outgoing mail server. This module will select the appropriate mail server automatically depending on who is sending the email. Therefore, all your emails will always be delivered without issues provided you have configured at least one outgoing mail server
